This plugin adds two extra buttons: 'Save & Add New' and 'Save & Close' to the edit
screen.

## Description

Add two extra buttons: 'Save & Add New' and 'Save & Close' to the edit screen.
'Save & Add New', publishes the item and redirects you to an Add New screen;
while 'Save & Close', publishes the item and redirects you to the edit screen
where you can see all the other items.

Original plugin 'Publish And Redirect To Add New Post' from Mindstien
(https://wordpress.org/plugins/publish-and-redirect-to-add-new-post/).

## Installation

Download, unzip and upload to wordpress
